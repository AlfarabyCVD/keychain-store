# Ganciku.id Web Design

Perubahan yang sudah dilakukan:
- Nama brand/app diganti dari KeyNova menjadi Ganciku.id.
- Title browser diganti menjadi: Ganciku.id — Gantungan Kunci Custom Futuristik.
- Meta description SEO diganti untuk brand Ganciku.id.
- Footer dan tombol WhatsApp ikut disesuaikan.
- package.json diganti menjadi ganciku-id.

Catatan penting:
Untuk mengganti alamat website dari keychain-store-beta.vercel.app menjadi ganciku.id, domain ganciku.id harus dibeli/aktif lalu ditambahkan di Vercel melalui menu Project Settings > Domains.
